# Git Cheatsheet

## Basic 

### Create empty repo
```
git init directory
```

### Clone repo
```
git clone repo
```

## Remote repositores

### Connection
```
git remote add <name> <url>
```

### Connection
```
git remote add <name> <url>
```

### Fetch a specific branch 
```
git fetch <remote> <branch>
```

### Pull remote 
```
git pull <remote>
```

### Push remote
```
git push <remote> <branch>

```
